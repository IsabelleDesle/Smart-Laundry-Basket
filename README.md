Smart Laundry Basket 

Deep learning is used to identify laundry in a laundry basket and classify it by washing program: 
- 30°C color wash
- 30°C white
- 60°C color
- 60°C white
- delicate and wool

The laundry basket has a camera with an infrared motion detector. 
Each time a motion is detected a photo of the content of the laundry basket is made and processed using deep learning. 

The count by washing program is tracked and an alert is shown if the count is high enough to fill and start the washing machine.
